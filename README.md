// Como correr en ambiente loca

* tienes que crear archivo '.env.development.local'

```
VITE_PROJECT=jwt
VITE_API_URL=http://localhost:8080/
VITE_API_VERSION=1.19.12.23+1
VITE_NOMBRE_PAGINA=''
VITE_PB_API_URL=http://localhost:8080/
```

* Correr Motor de Javascript

```
yarn install

yarn run dev

yarn run build
```
